Class ArrayShortListIterator

  • java.lang.Object
    • org.joda.primitives.listiterator.impl.ArrayShortListIterator
  • All Implemented Interfaces:
    Iterator<Short>, ListIterator<Short>, PrimitiveIterator<Short>, ShortIterator, PrimitiveListIterator<Short>, ShortListIterator

    public class ArrayShortListIterator
extends Object
implements ShortListIterator
    An iterator over an array of short values.

    This class implements ListIterator allowing seamless integration with other APIs.

    The iterator can be reset to the start if required. add() and remove() are unsupported, but set() is supported.
